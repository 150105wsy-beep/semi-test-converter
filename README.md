# 📊 半导体测试数据转换工具（CSV → TXT / MEA）

这是一个基于 **Streamlit** 的 Web 工具，用于将半导体器件测试产生的 CSV 数据文件，批量转换为特定测试系统可识别的 TXT / MEA 格式。

本项目以开源方式发布，旨在降低数据后处理的重复劳动成本，并鼓励社区协作与长期维护。

---

## ✨ 功能特性

- 支持单个或多个 CSV 文件批量上传  
- 自动从文件名解析器件参数（L、Vds）  
- 支持手动参数输入，并可强制覆盖自动解析结果  
- 自动生成 `.txt` 与 `.mea` 两种格式文件  
- 支持 curve 顺序反转，避免 Vbs 顺序混乱  
- 提供在线预览、单文件下载与 ZIP 批量下载  

---

## 📂 文件名解析规则

### L 参数
- 规则：提取 `L` 前的数字  
- 示例：  
  - `Ids_vgs_0.5L_vds0.05.csv` → `L = 0.5`  
  - `test_1.2L_vds1.8.csv` → `L = 1.2`  

### Vds 参数
- 规则：提取 `vds` 后的数字（不区分大小写）  
- 示例：  
  - `vds0.05` → `Vds = 0.05`  
  - `VDS1.8` → `Vds = 1.8`  

### W 参数
- 固定为 `1`（不从文件名解析）

当文件名不符合规范时，程序将自动使用手动输入的默认值。

---

## 🧠 转换逻辑说明

- 自动检测 CSV 文件编码（`utf-8 / gbk / gb2312 / latin-1`）
- 识别 CSV 中的 `"x","y"` 标识行，并依次替换为：

```text
curve { -1.8 }
curve { -1.35 }
curve { -0.9 }
curve { -0.45 }
curve { 0 }
````

* 支持一键反转 curve 顺序
* 数据行采用固定宽度排版，保证下游工具可正确解析

## 🚀 使用方式

### 在线使用

部署于 Streamlit Cloud，可直接通过浏览器访问。

### 本地运行

```bash
pip install streamlit pandas
streamlit run streamlit_app.py
```
